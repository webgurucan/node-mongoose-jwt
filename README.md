### <h1 align="center" id="heading">JWT Authentication Implementation in Node.js</h1>

1. Copy `.env.example` to `.env`
2. Update `CONNECTION_STRING`
3. Update `JWT_SECRET` Key 
4. `npm start`

Note:- You can use any random string as `JWT_SECRET`
